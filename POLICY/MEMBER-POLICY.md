# (2026-04-19) 할두 멤버쉽 정책서

**lv.0** SNS 간편 회원가입으로 가입만 한 상태
**lv.1** 필수 입력 항목 모두 작성한 상태 : *추천인 코드 매칭되지 않거나, 성별 M이거나, 생년월일 만 45세 이하일 경우 lv.2로 넘어가지 않음, 게시판 전체 열람 불가 (글의 첫줄들만 볼 수 있음)
**lv.2** 선택 입력 항목 기재 가능, 게시판 전체 열람 및 댓글 작성 가능
**lv.3** 스토어 상품 열람 및 구매 가능
**lv.4** 상품 작성 가능
**lv.5** 공지, 투표 작성 가능
