## 1. 등급 체계 (이원 구조)

### 1-1. 권한 등급 (어드민 관리용, 고객 비노출)

| 등급 | 상태 | 권한 | 등업 조건 |
|------|------|------|----------|
| **lv.0** | 간편 가입만 완료 | 앱 둘러보기만 가능 | 카톡으로 시작하기 |
| **lv.1** | 필수 정보 작성 완료 (심사 대기) | 게시판 제한 열람 (첫 문장만 노출, 나머지 흐림 처리) | 필수 입력 항목 모두 작성 |
| **lv.2** | 정회원 (승인 완료) | 게시판 전체 열람, 댓글·글쓰기 가능, 선택 입력 기재 가능 | 심사 조건 충족 + 관리자 승인 |
| **lv.3** | 스토어 이용 회원 | lv.2 권한 + 스토어 상품 열람·구매 | 관리자 수동 등업 |
| **lv.4** | 판매 회원 | lv.3 권한 + 상품 등록 가능 | 관리자 수동 등업 |
| **lv.5** | 운영 회원 | lv.4 권한 + 공지·투표 작성 가능 | 관리자 수동 등업 |

### 1-2. 활동 등급 (고객 노출, 활동량 기반)

lv.2 이상 정회원에게 적용되며, 월별 인증 활동량에 따라 자동 산정됨.

| 등급명 | 아이콘 | 조건 | 혜택 |
|--------|--------|------|------|
| **한잎 클로버** | 🍀 | 기본 (정회원 승인 시) | — |
| **두잎 클로버** | 🍀🍀 | 월 인증 80% 이상 | 크레딧 2만원, 초대장 1장 |
| **세잎 클로버** | 🍀🍀🍀 | 상위 10% | 크레딧 3만원, 초대장 2장 |
| **네잎 클로버** | 🍀🍀🍀🍀 | 상위 1% | 크레딧 5만원, 초대장 3장, 할두 클럽 개설권 |

## 2. 회원가입 흐름

### 가입 방식
**카톡으로 시작하기** (카카오 로그인 단일 수단)

### 필수 입력 (lv.0 → lv.1 등업 시)

| 항목 | 입력 방식 | 비고 |
|------|----------|------|
| 이름 | 텍스트 입력 | 실명 |
| 연락처 | 핸드폰 번호 입력 | SMS 인증 필수 |
| 닉네임 | 텍스트 입력 | 중복 확인 필수 |
| 생년월일 | 년/월/일 드롭다운 | 범위: 1952년~1982년, 초기값: 1960년 1월 1일 |
| 추천인 코드 | 텍스트 입력 | 형식: 추천인 닉네임 + 핸드폰 번호 뒷 4자리 |

> 생년월일 선택 항목 아래 안내 문구: **"45세부터 75세까지 가입 신청할 수 있어요"**

### lv.1 → lv.2 심사 조건

본인 인증(SMS) 정보를 기반으로 자동 판별 후, 관리자가 최종 승인함.

| 조건 | 기준 | 판별 방식 |
|------|------|----------|
| 추천인 코드 | 유효한 코드 매칭 | 시스템 자동 검증 |
| 성별 | 여성 | 본인 인증 정보로 판별 |
| 나이 | 만 45세 이상 ~ 75세 이하 | 생년월일 기준 자동 판별 |

**심사 미충족 시 안내 문구:** "아쉽지만 가입 조건이 맞지 않아요"

### lv.1 제한 열람 UX

- 글의 첫 문장은 전체 노출
- 나머지 내용은 **Blur(흐림) 처리**
- 흐린 영역 클릭 시 안내: **"정회원이 되면 볼 수 있어요"**
- 안내 아래 **[등업하기]** 버튼 노출

### 선택 입력 (lv.2 승인 후 기재 가능) — 10문 10답

| # | 질문 | 예시 |
|---|------|------|
| 1 | 취미 | 맛집 탐방 |
| 2 | 나의 장점 | 적극적이다 |
| 3 | 애창곡 | 아모르파티 |
| 4 | 좌우명 | 일단 해보자 |
| 5 | 인생 여행지 | 강원도 영월 |
| 6 | 좋아하는 운동 | 슬로우조깅 |
| 7 | 즐겨먹는 음식 | 찐 호박 |
| 8 | 나의 건강팁 | 스트레스 받지 않기! |
| 9 | 나의 5년 후 | 동네 화실 만들기 |
| 10 | 주 활동지역 | 서울 서대문 |

## 3. 추천인 코드 규칙

**형식:** `추천인 닉네임` + `핸드폰 번호 뒷 4자리`

**예시:** 슬기엄마 + 1452 → `슬기엄마1452`

- 추천인이 코드를 알려주거나, 카톡 공유 시 자동 입력
- 시스템에서 닉네임+번호 조합으로 매칭 검증

## 5. 등급 체계 관계도

```
[카톡으로 시작하기]
    │
    ▼
  lv.0 (간편 가입)
    │ 필수 정보 작성
    ▼
  lv.1 (심사 대기) ─── 게시판 제한 열람 (첫 문장만, 나머지 Blur)
    │ 심사 통과 + 관리자 승인
    ▼
  lv.2 (정회원) ────── 전체 열람, 댓글·글쓰기, 10문10답
    │                   ├── 한잎 클로버 🍀 (기본)
    │                   ├── 두잎 클로버 🍀🍀 (월 인증 80%+)
    │                   ├── 세잎 클로버 🍀🍀🍀 (상위 10%)
    │ 관리자 수동 등업    └── 네잎 클로버 🍀🍀🍀🍀 (상위 1%)
    ▼
  lv.3 (스토어 이용) ── 상품 열람·구매
    │ 관리자 수동 등업
    ▼
  lv.4 (판매 회원) ──── 상품 등록
    │ 관리자 수동 등업
    ▼
  lv.5 (운영 회원) ──── 공지·투표 작성
```
