# 프로젝트명: 할두 (HALDO)

## 1. 한 줄 소개
시니어 여성(5060+)이 건강, 살림 정보를 공유하고 믿을 수 있는 상품을 구매하는 '큰 글씨' 반응형 커뮤니티 & 마켓입니다.

---

## 2. 목표 사용자
- **주 사용층:** 디지털 기기 사용이 서툴지만 온라인 소통을 원하는 5060 여성
- **페르소나:** 작은 글씨에 피로감을 느끼며, 복잡한 결제 과정보다 문자 상담이나 간편 결제, 계좌 이체를 선호하는 사용자

---

## 3. 핵심 기능 요구사항 (Functional Requirements, FR)

### 3.1. 공통 (Common)
- [ ] **가독성 최적화 UI:** 전역 폰트 크기 조절 기능(A+, A-) 및 최소 18px 이상의 본문 사이즈 유지
- [ ] **반응형 웹:** Mobile-First 기반으로 태블릿 및 데스크톱 최적화.
- [ ] **심플 커뮤니티:** 사진 중심의 카드형 게시글 리스트 및 큰 글씨 댓글 작성 시스템
- [ ] **직관적 마켓:** 상품 상세 페이지 내 '문자 주문/상담' 플로팅 버튼 상시 노출
- [ ] **간편 로그인:** 카카오/네이버 소셜 계정을 활용한 복잡하지 않은 회원가입 절차


### 3.2. 홈 (Home)

- [ ]  **웰컴 보드:** 사용자 이름과 따뜻한 인사가 포함된 배너.
- [ ]  **퀵 메뉴:** 커뮤니티 베스트글, 마켓 추천 상품으로 바로가는 큰 카드형 버튼.
- [ ]  **~~실시간 날씨/건강 팁:** 시니어 관심사 위주의 데일리 정보 위젯.~~

### 3.3. 커뮤니티 (Community)

- [ ]  **카테고리 구성:** 건강/운동, 요리/살림, 취미/여행, 일상 대화.
- [ ]  **게시글 작성:**  글쓰기 버튼과 댓글 작성 버튼을 크게 배치.
- [ ]  **댓글 시스템:** 큰 글씨의 댓글 입력창과 이모티콘 지원.

### 3.4. 마켓 (Market)

- [ ]  **상품 목록:** 복잡한 필터 대신 큰 이미지와 단순한 가격 표기.
- [ ]  **상세 페이지:** '구매하기' 버튼 하단에 '문자로 상담하기' 버튼 상시 노출.
- [ ]  **간편 결제:** 페이류(카카오/네이버) 연동을 통한 무인증/간편 인증 결제 프로세스.

---

## 4. 추가 기능 (Nice-to-have)
- **안심 알림:** 내 글에 반응이 오면 직관적인 큰 아이콘과 함께 푸시 알림 제공

---

## 5. 기술 스택
- **Frontend:** Next.js (SEO 및 빠른 로딩), Tailwind CSS (반응형 및 고대비 테마)
- **Backend/DB:** Supabase (Auth, Database, Storage 통합 관리)
- **Infrastructure:** Vercel (배포 및 도메인 관리)

---

## 6. UI/디자인 방향
- **시각적 명확성:** 아이콘과 텍스트를 반드시 병행 표기 (예: 🏠 홈, 🛒 장바구니)
- **색상 팔레트:** 눈이 편안한 웜 톤(Beige, Soft Pink)과 명확한 대비를 이루는 포인트 컬러
- **레이아웃:** 모바일 우선(Mobile-First) 반응형 디자인으로 손가락 클릭 영역을 넓게 확보

## 7. 참고 레퍼런스
- **컬리(Kully):** 썸네일을 보고 상품담기, 상세 페이지 접속 시 이미지 출력
- **퀸잇(Queenit):** 시니어 여성 타겟의 커머스 레이아웃 및 폰트 활용 참고
- **카카오톡:** 전 국민에게 익숙한 하단 탭 바 및 알림 UI 패턴 차용

---

## 8. 제약 조건 / 주의사항
- **단순한 뎁스:** 모든 기능은 홈 화면에서 최대 3회 클릭 이내에 도달해야 함
- **실수 방지:** '삭제' 또는 '결제' 등 중요 액션 시 명확한 한국어 확인 모달 필수
- **용어 순화:** 어려운 영문 IT 용어 배제 (예: Feed -> 소통방, Mypage -> 내 정보)
